# hernia-repair

# Necessary Steps
First follow the instructions for the Biomedisa installation:
  Important Note!!!: Anaconda must be installt in the %Userprofile% directory!
  https://github.com/biomedisa/biomedisa/blob/master/README/windows10+cuda11.3+deeplearning_cli.md

When cloneing the biomedisa git repository also clone this repository.
Use the comand: 
    git clone https://github.com/biomedisa/hernia-repair.git
The repository should be in the directory:
  %Userprofile%\git (same as biomedisa)

# Installation of Matlab Runtime (Version R2019_b [9.7])
Install the Matlab Runtime from:
  https://de.mathworks.com/products/compiler/matlab-runtime.html?msclkid=51d2111fba4111ec85f6a9889cda0e5d

# Installtion of Paraview (Version 5.10.1 for Python 3.9)
Install Paraview in the same directory as Anaconda (%Userprofile%).
  https://www.paraview.org/download/
